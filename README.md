# Proyecto de Calidad de Software

Proyecto simple en Python para análisis de métricas estáticas usando **SonarCloud**.

Incluye:
- Funciones aritméticas
- Código con complejidad
- Pruebas unitarias

Este repositorio será analizado con SonarCloud para obtener:
- Complejidad ciclomática
- Duplicación
- Code Smells
- Bugs
- Cobertura de comentarios
